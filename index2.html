Summary

This site provides a report on the data analysis project investigating how weather changes on approaching the equator. Data was retrieved from OpenWeatherMap API to assemble a dataset of over 500 cities. The Matplotlib library was used to render plots for weather parameters as a function of the latitude. Visualization pages of plots for latitude versus temperature, cloudiness, wind speed, and humidity are shown.
